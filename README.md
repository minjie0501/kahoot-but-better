# Quiz Game

## Challenge
This exercise was part of the BeCode coding bootcamp. The goal was to create a quiz game similar to Kahoot.

- [x] A trivia style quiz
- [x] Setting (e.g. subject, difficulty, amount of questions)
- [x] Score
